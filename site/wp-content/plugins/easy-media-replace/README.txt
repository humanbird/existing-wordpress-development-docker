Replace Images and Media Files in WordPress Easily and Quickly. 

== Description ==


Easy Media Replace allows you to replace images and media files easily while retaining the attachment URL and ID. No more delete, re-upload, and search-replace.   


### Features

* Easy to use.
* Drag and drop file uploading.
* Replace images, PDF, videos, audios, and all media files.
* Regerenate thumbnails (no plugin required).
* Don't leave your page: Upload and replace files directly within current page.

### Usage

Once you activated the plugin, you will see a "Replace" button/link in Media Library page and dialog (see screenshots below).

== Installation ==
### From within WordPress

1. Visit ‘Plugins > Add New’.
2. Search for ‘Easy Media Replace’.
3. Install the plugin.
4. Activate the plugin.

### Manually

1. Upload the easy-media-replace folder to the /wp-content/plugins/ directory.
2. Activate the Easy Media Replace plugin through the ‘Plugins’ menu in WordPress.

== Frequently Asked Questions ==
= Can I replace images with different extensions =
Currently, the plugin only replaces images with same extension.
